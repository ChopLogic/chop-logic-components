# Changelog

All notable changes to this project will be listed in this file.

## [3.3.0](https://www.npmjs.com/package/chop-logic-components/v/3.3.0) (Oct 11, 2025)

- Added Link component with external and internal link support
- Added Breadcrumbs component with customizable items
- Updated documentation for Link and Breadcrumbs components
- Fixed SonarCloud issues

## [3.2.0](https://www.npmjs.com/package/chop-logic-components/v/3.2.0) (Oct 4, 2025)

- Added Search component with automatic and manual search modes
- Added useDebounce hook with interactive story
- Refactored Checkbox and Search components to use IconName enum for icons
- Updated documentation for Checkbox and Search components
- Upgraded Storybook to v9.1.10

## [3.1.0](https://www.npmjs.com/package/chop-logic-components/v/3.1.0) (Sept 26, 2025)

- Extend Tabs component by adding editable titles and dynamic tab management
- Updated documentation for Tabs component
- Fixed styles import issue
- Fixed SonarCloud issues

## [3.0.0](https://www.npmjs.com/package/chop-logic-components/v/3.0.0) (Sept 11, 2025)

- Fixed build script
- Optimized the package size
- Renamed all exported components by removing "ChopLogic..." prefix

## [2.6.0](https://www.npmjs.com/package/chop-logic-components/v/2.6.0) (Sept 6, 2025)

- Added a new component EditableText
- Refactored code structure
- Removed custom fonts in favor of browser default ones for compatibility
- Added a new story for the Label component
- Upgraded Storybook

## [2.5.3](https://www.npmjs.com/package/chop-logic-components/v/2.5.3) (Aug 9, 2025)

- Updated Introduction page
- Update publish to GitHub pages workflow
- Upgraded Storybook

## [2.5.2](https://www.npmjs.com/package/chop-logic-components/v/2.5.2) (Aug 6, 2025)

- Updated links in documentation files
- Updated npm publish workflow

## [2.5.0](https://www.npmjs.com/package/chop-logic-components/v/2.5.0) (Aug 4, 2025)

- Upgraded Storybook

## [2.4.1](https://www.npmjs.com/package/chop-logic-components/v/2.4.1) (Jul 27, 2025)

## [2.4.0](https://www.npmjs.com/package/chop-logic-components/v/2.4.0) (Jul 18, 2025)
