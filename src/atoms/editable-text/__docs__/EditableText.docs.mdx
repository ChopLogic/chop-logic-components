# EditableText

The `EditableText` component allows users to seamlessly edit inline content by clicking on it. It transitions from a read-only view to an editable input field.

## Usage

```tsx
import EditableText from 'chop-logic-components';

const UserProfile = () => {
  const [name, setName] = React.useState('John Doe');

  return (
    <div>
      <h2>
        <EditableText
          value={name}
          onChange={setName}
          placeholder="Enter your name"
        />
      </h2>
    </div>
  );
};
```

### When to Use

- Smooth transition between read and edit modes
- Ideal for titles, names, and short descriptions

### Accessibility Notes

1. _Keyboard Support_

    - Enter/Space to enter edit mode
    - Enter to save (single-line)
    - Escape to cancel edits
    - Tab to move focus

2. _Screen Readers_

    - Announces edit mode changes
    - Reads placeholder text
    - Indicates required fields

3. _Focus Management_

    - Auto-focuses when entering edit mode
    - Maintains proper focus order
    - Visible focus indicators

### Best Practices

    - Add validation in onChange handler
    - Provide visual feedback for invalid input
    - Memoize onChange handlers
    - Consider debouncing for frequent updates
