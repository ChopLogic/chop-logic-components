# Icon

The `Icon` component allows you to render one of the icons from the list as an SVG element. Icons can be inserted into inputs, buttons, menu items and any other React components.

## Iconography

<IconGallery>
  {Object.entries(IconName).map(([key, value]) => (
    <IconItem name={key} key={key}>
      <Icon name={value} />
    </IconItem>
  ))}
</IconGallery>

## Usage

```tsx
import { Icon, IconName } from 'chop-logic-components';

const MyButton = () => (
  <button>
    <Icon name={IconName.Delete} />
    <span>Delete</span>
  </button>
);

export default MyButton;
```

## Best Practices

1. _Basic usage_: Use when you need to draw attention to UI elements or when you need to replace a long text description with a clear picture.
2. _Accessibility_: When using icons, ensure they're purely decorative or provide additional context via ARIA attributes if they convey meaning.
