# ChopLogicIcon

The `ChopLogicIcon` component allows you to render one of the icons from the list as an SVG element. Icons can be inserted into inputs, buttons, menu items and any other React components.

## Iconography

<IconGallery>
  {Object.entries(ChopLogicIconName).map(([key, value]) => (
    <IconItem name={key}>
      <ChopLogicIcon name={value} />
    </IconItem>
  ))}
</IconGallery>

## Usage

```tsx
import React from 'react';
import { ChopLogicIcon, ChopLogicIconName } from 'chop-logic-components';

const MyButton = () => (
  <button>
    <ChopLogicIcon name={ChopLogicIconName.Delete} />
    <span>Delete</span>
  </button>
);

export default MyButton;
```


### When to Use

- When you need to draw attention to UI elements
- When you need to replace a long text description with a clear picture

### Accessibility Notes

When using icons, ensure they're purely decorative or provide additional context via ARIA attributes if they convey meaning.
