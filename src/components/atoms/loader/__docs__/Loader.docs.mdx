# Loader

The `Loader` component provides a collection of animated loading indicators to show progress or pending states in your application. It offers multiple visual styles to match different contexts and use cases.

## Usage

```tsx
import Loader, { LoaderView } from 'chop-logic-components';

const PageLoader = () => (
  <div>
    <h2>Content is loading...</h2>
    <Loader view={LoaderView.Circle} />
  </div>
);

export default PageLoader;
```

## Best Practices

1. _ARIA Attributes_: The component automatically includes role="status" and aria-live="polite" to announce loading states to screen readers.
2. _Color Contrast_: Ensure your loader has sufficient contrast against its background.
3. _Alternative Text_: For critical loading states, consider adding additional text explanation.
4. _Motion Reduction_: Respect prefers-reduced-motion media query in your CSS animations.
