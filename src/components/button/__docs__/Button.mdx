# ChopLogicButton

A simple button component with different props.

#### Example

<Canvas of={Button.Primary} />

## Usage

```ts
import { ChopLogicButton } from "chop-logic-components";

const Example = () => {
  return (
      <ChopLogicButton
        text="Submit"
        onClick={()=> console.log("Clicked")}
        type='submit'
        icon='CheckMark'
      />
  );
};

export default Example;
```

#### Props

- **view** - An appearance of the button. Can be `primary` (by default), `danger` (accented) or `secondary` (a clickable text).
- **text** - A string that represents the text content of the button (`Ok` by default).
- **onClick** _`() => void`_ - A function that is called when the button is clicked. It receives a MouseEventHandler for handling the click event on the button element. It could also be a callback without arguments.
- **disabled** - A boolean indicating whether the button should be disabled or not. When disabled, the button cannot be clicked or interacted with.
- **type** - A button type that can be `button`, `submit` or `reset` (`button` by default).
- **icon** - A name of a UTF-8 symbol for an icon, which could be inserted before the button text.
- **ButtonHTMLAttributes** - all standard HTML button attributes (**name**, **value**, **form**, etc.).
- **HTMLAttributes** - all global HTML attributes (**id**, **className**, **title**, **style**, etc.).
