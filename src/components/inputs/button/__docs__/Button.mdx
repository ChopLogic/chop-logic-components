# ChopLogicButton

Styled button component with different view modes and a customizable icon.

### Default View

<Canvas of={ButtonStories.DefaultButton} />

### Usage

```tsx
import { ChopLogicButton, ChopLogicIconName } from 'chop-logic-components';
import { createCopy } from 'utils/copy';

const Example = () => {
  return <ChopLogicButton text='Copy' onClick={createCopy} icon={ChopLogicIconName.Copy} />;
};

export default Example;
```

### Props

| Prop          | Type                                  | Description                                                                                                                                                                              | Default     | Required |
|---------------|---------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-------------|----------|
| **text**      | `string`                              | A string that represents the text content of the button.                                                                                                                                 | `undefined` | No       |
| **view**      | `ChopLogicButtonView`                 | An appearance of the button.                                                                                                                                                             | `undefined` | No       |
| **onClick**   | ` MouseEventHandler`                  | A function that is called when the button is clicked. It receives a MouseEventHandler for handling the click event on the button element. It could also be a callback without arguments. | `undefined` | No       |
| **disabled**  | `boolean`                             | A boolean indicating whether the button should be disabled or not. When disabled, the button cannot be clicked or interacted with.                                                       | `undefined` | No       |
| **type**      | `'button'` or `'reset'` or `'submit'` | The default behavior of the button.                                                                                                                                                      | `'button'`  | No       |
| **icon**      | `ChopLogicIconName`                   | An svg icon from ChopLogicIcon set, which could be inserted before the button text.                                                                                                      | `undefined` | No       |
| **id**        | `string`                              | The id global attribute defines an identifier (ID) which must be unique in the whole document.                                                                                           | `undefined` | No       |
| **className** | `string`                              | The class global attribute is a list of the classes of the element.                                                                                                                      | `undefined` | No       |
| **style**     | `CSSProperties`                       | The style global attribute contains CSS styling declarations to be applied to the element.                                                                                               | `undefined` | No       |
| **tabIndex**  | `number`                              | The tabindex global attribute allows developers to make HTML elements focusable.                                                                                                         | `undefined` | No       |
| **title**     | `string`                              | The title global attribute contains text representing advisory information related to the element it belongs to.                                                                         | `undefined` | No       |
| **theme**     | `'light'` or `'dark'`                 | The component's color theme.                                                                                                                                                             | `undefined` | No       |

### Types

```ts
export type ChopLogicButtonProps = Partial<CommonInputProps> & {
  onClick?: React.MouseEventHandler<HTMLButtonElement>;
  view?: ChopLogicButtonView;
  icon?: ChopLogicIconName;
  text?: string;
  extended?: boolean;
  type?: 'button' | 'reset' | 'submit';
};

export type ChopLogicButtonView = '3D' | 'flat' | 'icon';

export type CommonInputProps = CommonComponentProps & {
  label: string;
  name: string;
  disabled?: boolean;
  required?: boolean;
};

export type CommonComponentProps = {
  id?: string;
  className?: string;
  style?: React.CSSProperties;
  tabIndex?: number;
  title?: string;
  theme?: 'dark' | 'light';
};
```
