# ChopLogicCheckbox

Styled checkbox component with an optional label and floating checkbox icon. Can be used as input for ChopLogicForm.

### Default View

<Canvas of={CheckboxStories.LeftIconCheckbox} />

### Usage

```tsx
import { ChopLogicCheckbox } from 'chop-logic-components';

const Example = () => {
  return (
    <ChopLogicCheckbox
      id='terms-and-conditions-checkbox'
      name='isTermsAccepted'
      label='Accept Terms and Conditions'
      iconPosition='left'
      required
      onChange={(e) => console.log('Checkbox state:', e.target.checked)}
      disabled={false}
    />
  );
};

export default Example;
```

### Props

- **name** - A required string representing the name attribute of the Checkbox component, used for form submissions.
- **label** - A string that represents the label for the Checkbox component.
- **disabled** - A boolean indicating whether the checkbox should be disabled. When disabled, the checkbox cannot be interacted with.
- **required** - A boolean indicating whether the checkbox is required in a form. When required, the checkbox must be checked before form submission.
- **defaultChecked** - A boolean indicating whether the checkbox is checked by default or not.
- **onChange** - A function that is called when the checkbox state changes. It receives a change event.
- **noLabel** - A boolean indicating whether the label should be hidden or not. The label prop is still required in this case, because it's used for aria-label attribure.
- **iconPosition** - The place where the checkbox icon will be rendered (left or right).
- **HTMLAttributes** - additional global HTML attributes (**id**, **className**, **title**, **tabIndex**, **style** are supported, others are ignored).
