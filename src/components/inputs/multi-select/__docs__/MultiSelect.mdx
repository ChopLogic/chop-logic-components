# MultiSelect

MultiSelect component with custom styling and behavior.

#### Example

<Canvas of={MultiSelect.Default} />

## Usage

```tsx
import { ChopLogicMultiSelect } from 'chop-logic-components';
import { LANGUAGES } from 'constants/languages';

const Example = () => {
  return (
    <ChopLogicMultiSelect
      id='example-multiselect'
      name='languages'
      label='Select Languages'
      values={LANGUAGES}
      placeholder='Select languages'
      required={true}
      disabled={false}
    />
  );
};

export default Example;
```

#### Props

- **id** - A unique identifier for the MultiSelect component.
- **name** - A string representing the name attribute of the MultiSelect component, used for form submissions.
- **label** - A string that represents the label for the MultiSelect component.
- **values** - An array of objects representing the options available in the MultiSelect dropdown. Each object should have id, label, and selected properties.
- **placeholder** - A string that represents the placeholder text when no option is selected. Default is 'Not selected'.
- **required** - A boolean indicating whether the MultiSelect component is required in a form. When required, a selection must be made before form submission.
- **disabled** - A boolean indicating whether the MultiSelect component should be disabled. When disabled, the component cannot be interacted with.
- **onSelectChange** - A function that is called when an option is selected. It can be used to handle the selection event.
- **SelectHTMLAttributes** - all standard HTML select attributes (**name**, **value**, **form**, etc.). Note that not all of them are applicable to `ChopLogicMultiSelect`, because the component does not use default `<select>` tag.
- **HTMLAttributes** - all global HTML attributes (**id**, **className**, **title**, **style**, etc.).
