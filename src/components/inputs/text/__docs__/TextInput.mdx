# TextInput

TextInput component with different props.

#### Example

<Canvas of={TextInput.Default} />

## Usage

```tsx
import { TextInput } from 'sld-ui';

const Example = () => {
  return <ChopLogicTextInput label='Enter your name:' name='name' id='name-input' />;
};

export default Example;
```

#### Props

- **label** - A visible label that describes the input purpose.
- **name** - A string that describes the input for internal use inside a form or another container.
- **id** - A unique string that identifies the input on the page and connects it with the label.
- **disabled** - A boolean indicating whether the input should be disabled or not. When disabled, the input cannot be interacted with.
- **placeholder** - A string that represents the placeholder text inside the input field.
- **label** - A string that represents the label for the input field.
- **valid** - A boolean indicating whether the input is valid or not. When invalid, the input field may have a different style to indicate the error state.
- **required** - A boolean indicating whether the input is required or not.
