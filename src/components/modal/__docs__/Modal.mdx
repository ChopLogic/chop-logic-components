# Modal

Modal component with custom styling and behavior.

#### Example

<Canvas of={Modal.Default} />

## Usage

```tsx
import { ChopLogicModal } from 'chop-logic-components';
import { useState } from 'react';
import { MyModalContent } from 'components';

const Example = () => {
  const [isOpened, setIsOpened] = useState(true);

  const handleOpen = () => setIsOpened(true);
  const handleClose = () => setIsOpened(false);

  return (
    <>
      <button onClick={handleOpen}>Open Modal</button>
      <ChopLogicModal isOpened={isOpened} onClose={handleClose} title='Example Modal'>
        <MyModalContent />
      </ChopLogicModal>
    </>
  );
};

export default Example;
```

### Props

- **isOpened** - A boolean indicating whether the modal is open.
- **onClose** - A function that is called when the modal is closed.
- **title** - A string that represents the title of the modal.
- **children** - Contents of the modal window. Any React component enclosed between tags `<ChopLogicModal>` and `</ChopLogicModal>` will be rendered inside the modal using `ReactDOM.createPortal`.
- **HTMLAttributes** - all global HTML attributes (**id**, **className**, **title**, **style**, etc.).
