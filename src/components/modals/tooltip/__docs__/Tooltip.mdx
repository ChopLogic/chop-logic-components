# Tooltip

The `Tooltip` component displays a tooltip with customizable content when the user hovers over, clicks, or focuses on the container element.

<Canvas of={Tooltip.VisibleOnHover} />

### Usage

```tsx
import { ChopLogicTooltip } from 'chop-logic-components';

const TooltipUsageExample = () => {
  return (
    <ChopLogicTooltip tooltipContent='This is a tooltip' containerTag='span' visibleOn='hover' id='tooltip'>
      <span>Hover over me</span>
    </ChopLogicTooltip>
  );
};

export default TooltipUsageExample;
```

### Props

| Prop               | Type                                                   | Description                                                                                    | Default     | Required |
|--------------------|--------------------------------------------------------|------------------------------------------------------------------------------------------------|-------------|----------|
| **tooltipContent** | `string` or `ReactElement`                             | The content to be displayed inside the tooltip.                                                | `undefined` | Yes      |
| **containerTag**   | `TooltipContainerTag`                                  | The HTML tag to be used for the container element.                                             | `'span'`    | No       |
| **visibleOn**      | `'hover'` or `'click'` or `'focus'` or `'contextmenu'` | The event that triggers the tooltip to be displayed.                                           | `'hover'`   | No       |
| **children**       | `ReactNode`                                            | The content inside the container element that triggers the tooltip.                            | `undefined` | No       |
| **id**             | `string`                                               | The id global attribute defines an identifier (ID) which must be unique in the whole document. | `undefined` | No       |
| **className**      | `string`                                               | The class global attribute is a list of the classes of the element.                            | `undefined` | No       |
| **style**          | `CSSProperties`                                        | The style global attribute contains CSS styling declarations to be applied to the element.     | `undefined` | No       |
| **tabIndex**       | `number`                                               | The tabindex global attribute allows developers to make HTML elements focusable.               | `undefined` | No       |
| **theme**          | `'light'` or `'dark'`                                  | The component's color theme.                                                                   | `undefined` | No       |

### Types

```ts
export type ChopLogicTooltipProps = React.PropsWithChildren &
  CommonComponentProps & {
    tooltipContent: string | React.ReactElement;
    containerTag?: TooltipContainerTag;
    visibleOn?: 'hover' | 'click' | 'focus' | 'contextmenu';
  };

export type TooltipContainerTag = 'span' | 'div' | 'p' | 'strong' | 'em';

export type CommonComponentProps = {
  id?: string;
  className?: string;
  style?: React.CSSProperties;
  tabIndex?: number;
  title?: string;
  theme?: 'dark' | 'light';
};
```
