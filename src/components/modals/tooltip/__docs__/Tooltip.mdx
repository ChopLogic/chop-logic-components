# Tooltip

The `Tooltip` component displays a tooltip with customizable content when the user hovers over, clicks, or focuses on the container element.

#### Example

<Canvas of={Tooltip.VisibleOnHover} />

### Usage

```tsx
import { ChopLogicTooltip } from 'chop-logic-components';

const TooltipUsageExample = () => {
  return (
    <ChopLogicTooltip tooltipContent='This is a tooltip' containerTag='span' visibleOn='hover' id='tooltip'>
      <span>Hover over me</span>
    </ChopLogicTooltip>
  );
};

export default TooltipUsageExample;
```

### Props

_Required props_

- **tooltipContent** `string | React.ReactElement` - The content to be displayed inside the tooltip.

_Optional props_

- **containerTag** `TooltipContainerTag` - The HTML tag to be used for the container element. Default: `'span'`
- **visibleOn** `'hover' | 'click' | 'focus' | 'contextmenu'` - The event that triggers the tooltip to be displayed. Default: `'hover'`
- **children** `React.ReactNode` - The content inside the container element that triggers the tooltip.
- **HTMLAttributes** - some global HTML attributes (**id**, **className**, **title**, **style**, etc.).

### Types

```ts
type ChopLogicTooltipProps = PropsWithChildren &
  React.HTMLAttributes<HTMLElement> & {
    tooltipContent: string | React.ReactElement;
    containerTag?: TooltipContainerTag;
    visibleOn?: 'hover' | 'click' | 'focus' | 'contextmenu';
  };
```

```ts
type TooltipContainerTag = 'span' | 'div' | 'p' | 'strong' | 'em';
```
