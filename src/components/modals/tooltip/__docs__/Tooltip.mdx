# Tooltip

Tooltip component with custom styling and behavior.

#### Example

<Canvas of={Tooltip.Default} />

## Usage

```tsx
import { ChopLogicTooltip } from 'chop-logic-components';
import { useState } from 'react';

const Example = () => {
  const [isOpened, setIsOpened] = useState(true);

  const handleOpen = () => setIsOpened(true);
  const handleClose = () => setIsOpened(false);

  return (
    <>
      <button onClick={handleOpen}>Open Dialog</button>
      <ChopLogicTooltip isOpened={isOpened} onClose={handleClose}>
        <div>Some text</div>
      </ChopLogicTooltip>
    </>
  );
};

export default Example;
```

### Props

- **isOpened** - A boolean indicating whether the dialog is open.
- **onClose** - A function that is called when the dialog is closed.
- **title** - A string that represents the title of the dialog.
- **children** - Contents of the dialog window. Any React component enclosed between tags `<ChopLogicDialog>` and `</ChopLogicDialog>` will be rendered inside the dialog using `ReactDOM.createPortal`.
- **HTMLAttributes** - all global HTML attributes (**id**, **className**, **title**, **style**, etc.)../TooltipExample.tsx./Tooltip.stories.tsx
