# NumericInput

The `NumericInput` component provides a controlled numeric input field with spin buttons, validation, and accessibility features.

## Usage

```tsx
import NumericInput from 'chop-logic-components';

const QuantitySelector = () => {
  const [quantity, setQuantity] = React.useState(1);

  return (
    <NumericInput
      name="quantity"
      label="Quantity"
      min={1}
      max={10}
      step={1}
      defaultValue={1}
      onChange={(value) => setQuantity(value)}
    />
  );
};
```

## Best Practices

1. _Validation_:
   - Always provide errorMessage when using validation
   - Combine built-in min/max with custom validation when needed
   - Display validation errors clearly
2. _Value Ranges_:
   - Set sensible defaults for min/max
   - Consider the expected range of values
   - Adjust step size appropriately for the context
3. _Accessibility Notes_:
   - Use label prop for clear identification
   - Ensure spin buttons are keyboard accessible
   - Provide meaningful error messages for screen readers
