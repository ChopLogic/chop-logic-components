# Form

The `Form` component provides a structured, accessible form container with built-in state management, validation, and responsive layout capabilities.

## Usage

```tsx
import { Form, TextInput } from "chop-logic-components";

const UserForm = () => {
  const handleSubmit = (values) => {
    console.log("Form submitted with:", values);
  };

  return (
    <Form initialValues={{ firstName: "", lastName: "" }} onSubmit={handleSubmit}>
      <TextInput name="firstName" label="First Name" />
      <TextInput name="lastName" label="Last Name" />
    </Form>
  );
};
```

## Best Practices

1. _Layout_:
   - Use appropriate column counts for form complexity
   - Group related fields together
   - Maintain consistent spacing
2. _Validation_:
   - Provide clear error messages
   - Validate on blur and submit
   - Consider real-time validation for complex forms
3. _Performance_:
   - Memoize form components when possible
   - Consider dynamic loading for large forms
   - Optimize re-renders with controlled components
4. _Accessibility_:
   - Ensure all form controls are keyboard accessible
   - Use ARIA attributes for dynamic content updates
   - Test with screen readers
