# useIsOverflow

The `useIsOverflow` hook detects when content overflows its container in either width or height dimension. Helps manage overflow scenarios by:
- Detecting content overflow dynamically
- Supporting both width and height dimensions
- Providing reactive state for overflow conditions
- Simplifying overflow-related UI logic

## Usage

```tsx
import { useRef } from 'react';
import { useIsOverflow } from 'chop-logic-components';

const OverflowComponent = () => {
  const containerRef = useRef<HTMLDivElement>(null);
  const isWidthOverflow = useIsOverflow({ref: containerRef, dimension: 'width'});

  return (
    <div ref={containerRef} className={isWidthOverflow ? 'has-overflow' : ''}>
      {/* Long content that might overflow */}
    </div>
  );
};
```

## Parameters

| Parameter     | Type                     | Description                                                                 |
|---------------|--------------------------|-----------------------------------------------------------------------------|
| **ref**       | `RefObject<HTMLElement>` | React ref object pointing to the element to monitor                         |
| **dimension** | `'width' \| 'height'`    | Which dimension to check for overflow                                       |
| **isMounted** | `boolean`    | Optional flag to trigger the hook on mount the parent component                                       |

## Returns

Returns a boolean state indicating whether overflow exists (`true`) or not (`false`).
