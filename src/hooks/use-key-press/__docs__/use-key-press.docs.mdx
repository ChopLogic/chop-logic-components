# useKeyPress

The `useKeyPress` hook listens for a specific key press event and calls a callback function when the key is pressed.

### Usage

```tsx
import React, { useRef } from 'react';
import { useKeyPress } from 'chop-logic-components';

const ExampleComponent = () => {
  const ref = useRef(null);

  const handleKeyPress = () => {
    console.log('Key pressed!');
  };

  useKeyPress({ ref, keyCode: 'Enter', onKeyPress: handleKeyPress });

  return (
    <div ref={ref} tabIndex={0}>
      Press "Enter" key
    </div>
  );
};

export default ExampleComponent;
```

### Params

| Param          | Type                           | Description                                                                                            |
|----------------|--------------------------------|--------------------------------------------------------------------------------------------------------|
| **ref**        | `React.RefObject<HTMLElement>` | A ref to the element that will be used to check if the key press event is within the element's bounds. |
| **keyCode**    | `string`                       | The key code of the key to listen for.                                                                 |
| **onKeyPress** | `() => void`                   | The callback function to call when the key is pressed.                                                 |


### Returns

`void`
