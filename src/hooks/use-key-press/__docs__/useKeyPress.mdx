# useKeyPress

The `useKeyPress` hook listens for a specific key press event and calls a callback function when the key is pressed.

<Canvas>
  <Story of={UseKeyPressStories.Default} />
</Canvas>

## Usage

```tsx
import React, { useRef } from 'react';
import { useKeyPress } from 'your-library';

const ExampleComponent = () => {
  const ref = useRef(null);

  const handleKeyPress = () => {
    console.log('Key pressed!');
  };

  useKeyPress({ ref, keyCode: 'Enter', onKeyPress: handleKeyPress });

  return (
    <div ref={ref} tabIndex={0}>
      Press "Enter" key
    </div>
  );
};

export default ExampleComponent;
```

##Params

- **ref**
  - Type: `React.RefObject<HTMLDivElement>`
  - Description: A ref to the element that will be used to check if the key press event is within the element's bounds.
- **keyCode**
  - Type: string
  - Description: The key code of the key to listen for.
- **onKeyPress**
  - Type: `() => void`
  - Description: The callback function to call when the key is pressed.
