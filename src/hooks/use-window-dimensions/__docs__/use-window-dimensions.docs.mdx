# useWindowDimensions

The `useWindowDimensions` hook tracks and provides the current window dimensions.

### Usage

```tsx
import { useWindowDimensions } from 'chop-logic-components';

const ExampleComponent = () => {
  const { width, height } = useWindowDimensions();

  return (
    <div>
      <p>Window width: {width}px;</p>
      <p>Window height: {height}px;</p>
    </div>
  );
};

export default ExampleComponent;
```

### Params

None.

### Returns

**windowDimensions** `{ width: number; height: number }`: An object containing the current window width and height.
