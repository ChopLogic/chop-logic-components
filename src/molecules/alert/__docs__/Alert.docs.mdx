# Alert

A flexible alert component that displays contextual feedback messages to users. Supports multiple modes (info, success, warning, error) with automatic closing, custom icons, and customizable timing.

## Usage

```tsx
import Alert, { AlertMode } from 'chop-logic-components';

const NotificationSystem = () => {
  const [alertOpen, setAlertOpen] = React.useState(true);

  return (
    <Alert
      isOpened={alertOpen}
      onClose={() => setAlertOpen(false)}
      mode={AlertMode.Success}
      title="Action completed"
      message="Your changes have been saved successfully."
      autoClose
    />
  );
};
```

## Best Practices

1. _Contextual Modes_: Use appropriate alert modes to convey the message importance.
2. _Auto-close Timing_: Set appropriate autoCloseDelay values based on message importance (shorter for info, longer for errors).
3. _Icon Selection_: Choose icons that reinforce the message type and improve visual recognition.
4. _Content Clarity_: Provide clear, concise titles and descriptive messages.
5. _Accessibility_: Ensure proper color contrast and include ARIA labels for different alert types.
6. _User Control_: Allow users to dismiss important alerts manually when autoClose is disabled.
