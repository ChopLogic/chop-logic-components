# ChopLogicTextInput

The `ChopLogicTextInput` component provides a flexible, accessible text input field with validation, clear button, and password visibility toggle.

## Usage

```tsx
import ChopLogicTextInput from 'chop-logic-components';

const LoginForm = () => {
  const [email, setEmail] = React.useState('');

  return (
    <ChopLogicTextInput
      name="email"
      label="Email address"
      type="email"
      placeholder="Enter your email"
      value={email}
      onChange={(e) => setEmail(e.target.value)}
      required
    />
  );
};
```

### Accessibility Notes

_Keyboard Navigation_:
  - Tab to focus the input
  - Clear button accessible via keyboard
  - Password toggle accessible via keyboard

_Screen Readers_:
  - Announces label and required state
  - Announces validation errors
  - Describes clear button action

_Focus Management_:
  - Clear focus states for all interactive elements
  - Maintains focus during interactions

### Best Practices

_Labeling_:
  - Always provide a meaningful label
  - Use placeholder text sparingly (not as a label replacement)
  - Ensure proper label-input association

_Validation_:
  - Provide clear error messages
  - Validate on blur as well as submit
  - Consider real-time validation for long forms

_Mobile Considerations_:
  - Use appropriate input types for virtual keyboards
  - Ensure touch targets are large enough
  - Test with screen magnifiers
