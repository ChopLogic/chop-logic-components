# TextInput

The `TextInput` component provides a flexible, accessible text input field with validation, clear button, and password visibility toggle.

## Usage

```tsx
import TextInput from 'chop-logic-components';

const LoginForm = () => {
  const [email, setEmail] = React.useState('');

  return (
    <TextInput
      name="email"
      label="Email address"
      type="email"
      placeholder="Enter your email"
      value={email}
      onChange={(e) => setEmail(e.target.value)}
      required
    />
  );
};
```

## Best Practices

1. _Labeling_:
   - Always provide a meaningful label
   - Use placeholder text sparingly (not as a label replacement)
   - Ensure proper label-input association
2. _Validation_:
   - Provide clear error messages
   - Validate on blur as well as submit
   - Consider real-time validation for long forms
3. _Accessibility_:
   - Ensure clear focus states for all interactive elements
   - Ensure Clear and Password Toggle buttons are keyboard accessible
   - Use ARIA attributes for dynamic content updates
