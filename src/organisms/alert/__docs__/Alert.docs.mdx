# ChopLogicAlert

The `ChopLogicDialog` component provides an accessible modal dialog with focus management, keyboard navigation, and flexible content structure.

## Usage

```tsx
import ChopLogicAlert, { AlertMode } from 'chop-logic-components';

const NotificationSystem = () => {
  const [alertOpen, setAlertOpen] = React.useState(true);

  return (
    <ChopLogicAlert
      isOpened={alertOpen}
      onClose={() => setAlertOpen(false)}
      mode={AlertMode.Success}
      title="Action completed"
      message="Your changes have been saved successfully."
      autoClose
    />
  );
};
```

### Accessibility Notes

_Focus Management_:
  - Focus returns to trigger element when closed
  - Close button is keyboard accessible

_Screen Readers:_
  - Announces alert content when appears
  - Announces severity level
  - Provides clear close action

_Color Contrast_:
  - Ensure sufficient contrast for all modes
  - Test in high contrast modes
  - Provide alternative indicators for color-blind users

### Best Practices

_Content Guidelines_:
  - Keep titles short (1-5 words)
  - Limit messages to 1-2 sentences
  - Use clear, actionable language

_Timing_:
  - Use auto-close for non-critical alerts
  - Allow sufficient reading time (minimum 5s)
  - Consider pause-on-hover for auto-closing alerts
