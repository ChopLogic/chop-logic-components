# ChopLogicForm

The `ChopLogicForm` component provides a structured, accessible form container with built-in state management, validation, and responsive layout capabilities.

## Usage

```tsx
import { ChopLogicForm, TextInput } from 'chop-logic-components';

const UserForm = () => {
  const handleSubmit = (values) => {
    console.log('Form submitted with:', values);
  };

  return (
    <ChopLogicForm
      initialValues={{ firstName: '', lastName: '' }}
      onSubmit={handleSubmit}
      columns={2}
    >
      <TextInput name="firstName" label="First Name" />
      <TextInput name="lastName" label="Last Name" />
    </ChopLogicForm>
  );
};
```

### Accessibility Notes

_Form Structure_:
- Proper form element semantics
- Field labels associated with inputs
- Logical tab order

_Validation_:
- Clear error messages
- ARIA attributes for invalid fields
- Focus management for errors

_Keyboard Navigation_:
- Tab through form fields
- Enter to submit
- Escape to reset (when enabled)

### Best Practices

_Layout_:
- Use appropriate column counts for form complexity
- Group related fields together
- Maintain consistent spacing

_Validation_:
- Provide clear error messages
- Validate on blur and submit
- Consider real-time validation for complex forms

_Performance_:
- Memoize form components when possible
- Consider dynamic loading for large forms
- Optimize re-renders with controlled components
