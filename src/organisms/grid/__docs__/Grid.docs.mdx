# Grid

The `Grid` component provides a flexible, interactive data table with column definitions, selection capabilities, and customizable rendering.

### Usage

```tsx
import Grid from 'chop-logic-components';

const UserGrid = () => {
  const columns = [
    { field: 'name', title: 'Name' },
    { field: 'email', title: 'Email' },
    { field: 'role', title: 'Role' }
  ];

  const data = [
    { id: '1', name: 'John Doe', email: 'john@example.com', role: 'Admin' },
    { id: '2', name: 'Jane Smith', email: 'jane@example.com', role: 'User' }
  ];

  return (
    <Grid
      columns={columns}
      data={data}
      caption="User Management"
      selectable
      onSelect={(ids) => console.log('Selected IDs:', ids)}
    />
  );
};
```

### Accessibility Notes

_Structure_:
- Proper table semantics
- Caption for context
- Column headers associated with cells

_Keyboard Navigation_:
- Tab through interactive elements
- Space to select rows
- Shift+Arrow for multiple selection

_Screen Readers_:
- Announces row selection state
- Reads column headers with cell values
- Provides context for actions

### Best Practices

_Performance_:
- Virtualize large datasets
- Memoize custom renderers
- Consider pagination for huge datasets

_Column Design_:
- Keep column count reasonable
- Use descriptive headers
- Highlight important columns

_Selection_:
- Provide clear visual feedback
- Consider bulk actions for selected items
- Disable non-applicable rows
