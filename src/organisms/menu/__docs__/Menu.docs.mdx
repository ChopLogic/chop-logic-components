# ChopLogicMenu

The component allows you to render a fully styled and accessible navigation menu in two modes - horizontal or vertical. Menu items can be described as a JavaScript array, and each item can contain nested items that will be rendered recursively. Nesting levels are not limited.

## Usage

```tsx
import React from 'react';
import { ChopLogicMenu, ChopLogicOrientationMode } from 'chop-logic-components';
import { MENU_ITEMS } from 'constants';

const ExampleComponent = () => (
  <ChopLogicMenu items={MENU_ITEMS} mode={ChopLogicOrientationMode.Horizontal} openedOn='click' />
);

export default ExampleComponent;
```
