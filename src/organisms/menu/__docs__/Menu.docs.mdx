# Menu

The `Menu` component provides an accessible navigation menu with support for both vertical and horizontal orientations, nested items, and flexible interaction modes.

## Usage

```tsx
import Menu, { OrientationMode } from 'chop-logic-components';

const AppMenu = () => {
  const menuItems = [
    {
      id: 'home',
      label: 'Home',
      link: '/home',
      icon: IconName.Home,
    },
    {
      id: 'products',
      label: 'Products',
      nestedItems: [{ id: 'product-1', label: 'Product 1', link: '/products/1' }],
    },
  ];

  return <Menu items={menuItems} mode={OrientationMode.Horizontal} openedOn="hover" />;
};
```

## Best Practices

1. _Organization_:
   - Limit nesting depth (recommend max 2-3 levels)
   - Group related items together
   - Sort items logically
2. _Interaction_:
   - Use hover activation for quick access
   - Use click activation for complex menus
   - Provide visual feedback for active items
3. _Performance_:
   - Memoize menu items when possible
   - Consider dynamic loading for large menus
4. _Keyboard Navigation_:
   - Tab to navigate through menu items
   - Enter/Space to activate items
   - Arrow keys to navigate nested menus
   - Escape to close submenus
