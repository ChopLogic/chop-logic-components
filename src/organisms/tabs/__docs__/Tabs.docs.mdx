# Tabs

The `Tabs` component provides an accessible tabbed interface with support for both horizontal and vertical orientations, disabled states, and flexible content.

### Usage

```tsx
import Tabs, { OrientationMode } from 'chop-logic-components';

const ProductTabs = () => {
  const tabs = [
    {
      id: 'details',
      title: 'Details',
      content: <ProductDetails />
    },
    {
      id: 'specs',
      title: 'Specifications',
      content: <ProductSpecs />
    }
  ];

  return (
    <Tabs
      tabs={tabs}
      mode={OrientationMode.Horizontal}
      defaultTabId="specs"
    />
  );
};
```

### Accessibility Notes

_Keyboard Navigation_:
- Tab to focus tab list
- Arrow keys to navigate between tabs
- Enter/Space to select focused tab

_Screen Readers_:
- Announces selected state
- Describes tablist structure
- Indicates disabled tabs

_Focus Management_:
- Focus remains on active tab
- Proper focus indicators
- Logical tab order

### Best Practices

_Organization_:
- Keep tab titles concise (1-2 words)
- Group related content together
- Limit number of tabs (recommend max 5-7)

_Content_:
- Load heavy content dynamically
- Consider lazy-loading hidden tabs
- Maintain consistent tab heights

_States_:
- Clearly indicate active tab
- Visually distinguish disabled tabs
- Provide loading states for async content
